## 프로젝트 개발 이유

폭설로 인해 교통 마비, 제설차 이동이 어려운 곳의 미끄러움으로 인한 피해 등등 때문에 생기는 사고를 최소화하고자 함.
이동의 제약이 없는 드론을 사용해서 제설차가 가기 어려운 곳도 제설작업을 수월하게 할 수 있도록 하는 것.

## 프로젝트 개요

### 주요 기능:
- **기능 1**: 드론 카메라 화면으로 눈을 인식.

- **기능 3**: 알림을 받은 드론은 제설제프로젝트 소개
